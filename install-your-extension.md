# Install Your Extension

## Add It To Your Profile[​](#add-it-to-your-profile "Direct link to heading")

Now that you have a manifest file we can install your extension in Owlbear Rodeo.

To do that you will need to create a new extension in your Owlbear Rodeo profile and point it to your new extension.

In your [profile](https://owlbear.app/profile) click the Add Extension button and use your manifest as the install link:

`http://localhost:5173/manifest.json`

![Extension install](/assets/images/install-99745dc6f0fe998a86babc6a2ce98a41.jpg)

## Create a Room[​](#create-a-room "Direct link to heading")

Next create a new room and enable your extension in the Create Room dialog.

![Select extension](/assets/images/select-1bd8c2f9c55ac0e51cb3e200a02c05f0.jpg)

Now you will be able to open your room and see your new extension available in the top left.

![Extension Action](/assets/images/action-5c7dc26dc6615729893f1b1879a4e69a.jpg)
